Xin Calendar 2 Mods: Date Info

yxScripts.com

 

The Date Info mod lets us embed date related contents into date cells.

To turn it on:

xcMods=[...
        {"order": 1,  "mod": "Date Info",  "script": "mod_info.js"},
        ...

or:

<script language="javascript" src="../config/xc2_default.js"></script>
<script language="javascript">
  xcMods[3].order=1;
</script>

To define the contents for dates, we have:

  • addDateInfo(conf_name, css, css_disabled, css_info, date_format, date, info, mode)

    1. conf_name ... a name for the configuration reference
    2. css ... css class (or two classes separated by a colon) for dates with contents defined, if empty, use the ones defined by [ xcCSSDaySpecial ] and [ xcCSSDayOtherSpecial ] in the config file
    3. css_disabled ... css class (or two classes separated by a colon) for dates with contents defined that are disabled, if empty, use the one defined by [ xcCSSDaySpecialDisabled ] and [ xcCSSDayOtherSpecial ]
    4. css_info ... css class for date content
    5. date_format ... date format used for following date parameter, if empty, use the one defined by [ xcDateFormat ]
    6. date ... date string (the year portion will be ignored for repeating mode)
    7. info ... date content in plain TEXT or HTML
    8. mode ... 1:one-time only, 0 (or absent):repeat yearly
