Xin Calendar 2 Mods: Date Link

yxScripts.com

The Date Link mod will open pages defined for dates.

To turn it on:

xcMods=[...
        {"order": 1,  "mod": "Date Link",  "script": "mod_link.js"},
        ...

or:

<script language="javascript" src="../config/xc2_default.js"></script>
<script language="javascript">
  xcMods[5].order=1;
</script>

To define page URLs with dates, we have:

  • addDateLink(conf_name, css, css_disabled, date_format, date, url)

    1. conf_name ... a name for the configuration reference
    2. css ... css class (or two classes separated by a colon) for dates with page URL defined, if empty, use the ones defined by [ xcCSSDaySpecial ] and [ xcCSSDayOtherSpecial ] in the config file
    3. css_disabled ... css class (or two classes separated by a colon) for dates with page URL defined that are disabled, if empty, use the ones defined by [ xcCSSDaySpecial ] and [ xcCSSDayOtherSpecial ]
    4. date_format ... date format used for following date parameter, if empty, use the one defined by [ xcDateFormat ]
    5. date ... date string
    6. url ... page URL
    addDateLink("conf", "day_current", "", "", "2004-08-01", "http://www.google.com");
addDateLink("conf", "day_current", "", "", "2004-08-31", "http://www.yahoo.com");


The Date Link mod shares some settings with the Journal mod in the default config file (config/xc2_default.js):

xcLinkBasePath="";
xcLinkTargetWindow="_blank";
xcLinkTargetWindowPara="location=1,menubar=1,resizable=1,scrollbars=1,status=1,titlebar=1,toolbar=1";
